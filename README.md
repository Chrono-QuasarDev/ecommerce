# Custom Designs Website

A landing page for a local custom printing business specializing in t-shirts, hoodies, stickers, wristbands, and more.

## Disclaimer

The initial boilerplate for this project was generated with the assistance of AI (Claude). All subsequent development, customizations, and improvements are done by me.

## Features

- Responsive landing page
- WhatsApp integration for direct messaging
- WhatsApp group invite link
- Product/services showcase
- Gallery of past work
- Mobile-friendly design

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
